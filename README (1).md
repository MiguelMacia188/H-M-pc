# H&M Energy — Sistema de Cotizaciones

Aplicación web para generar cotizaciones y facturas de venta de **H&M Energy Solutions & Products**.

## 🚀 Despliegue en GitHub Pages

### Paso 1: Crear el repositorio
1. Ve a [github.com](https://github.com) e inicia sesión
2. Haz clic en **"New repository"** (botón verde)
3. Nombre: `hym-cotizaciones` (o el que prefieras)
4. Marca **"Public"** (requerido para GitHub Pages gratuito)
5. Haz clic en **"Create repository"**

### Paso 2: Subir el archivo
1. En el repositorio nuevo, haz clic en **"uploading an existing file"**
2. Arrastra el archivo `index.html` y suéltalo ahí
3. Clic en **"Commit changes"**

### Paso 3: Activar GitHub Pages
1. Ve a **Settings** (pestaña del repositorio)
2. En el menú izquierdo, busca **"Pages"**
3. En "Branch", selecciona **`main`** y carpeta **`/ (root)`**
4. Haz clic en **"Save"**
5. Espera 1-2 minutos y tu URL será:
   `https://TU_USUARIO.github.io/hym-cotizaciones/`

## ✨ Características

- 📋 Cotizaciones y facturas con diseño profesional
- 🗄️ Base de datos de 57 materiales/servicios **editable**
- ➕ Agregar, modificar y eliminar ítems de la BD
- 🔍 Búsqueda instantánea de ítems
- 💰 Aplicar IVA (19%) opcional
- ⬇️ Descarga en PDF de alta calidad
- 💾 Guarda automáticamente en el navegador
- 📱 Interfaz responsive

## 📱 Uso
1. Llena los datos del cliente
2. Busca y agrega ítems (los precios son editables)
3. Ve a "Vista Previa" para revisar
4. Descarga el PDF
